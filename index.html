I'm a backend developer in training based in Spain. I'm doing a vocational degree in Software Development (DAM — a 2-year technical program focused on software engineering) and before that I did one in IT Systems & Networking (SMR — networks, servers, and operating systems from scratch). That gave me a solid foundation before I even started writing application code.

I work with Java and Spring Boot. That's my main thing — REST APIs, layered architecture, real projects I actually deploy and use. I'm not just following tutorials; I build things because I want to see them work.

On the side I'm learning Rust. Not because it's practical right now, but because it's genuinely fascinating — the way it thinks about memory and ownership is unlike anything else I've touched. It's the kind of challenge I enjoy.

I'm determined, consistent, and I know what I want: to make a living doing something that challenges me constantly. That's what this is about.

Outside of code — books, anime, gaming, walking, and learning things I definitely don't need yet.
