I'm a backend developer in training based in Spain. I'm doing a vocational degree in Software Development (DAM — a 2-year technical program focused on software engineering) and before that I did one in IT Systems & Networking (SMR — networks, servers, and operating systems from scratch).

The networking and systems background was unplanned but it stuck — it gives me context that I find useful when working on backend.

Java and Spring Boot is where I spend most of my time — REST APIs, layered architecture, security, persistence. Real projects I actually deploy and use, not just tutorials I followed once and forgot.

On the side I'm learning Rust. Not because it's practical right now, but because it's genuinely fascinating — the way it thinks about memory and ownership is unlike anything else I've touched. It's the kind of challenge I enjoy.

I'm determined, consistent, and I know what I want: to make a living doing something that challenges me constantly. That's what this is about.

Outside of code — books, anime, gaming, training and learning things I definitely don't need yet.
